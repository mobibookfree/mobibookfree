Customizing the page

Page Structure

The page is structured into 3 main divs, this structure must be followed to the letter to make the page work properly. 

<div class="all-elements">
    <div class="content-header">
        <a href="#" class="deploy-sidebar"></a>
        <img class="logo" src="images/misc/logo.png" width="92" alt="img">
        <a href="contact.html" class="deploy-contact"></a>
        <div class="clear"></div>
    </div>
    <div id="sidebar" class="page-sidebar">
        <div class="page-sidebar-scroll">
           <!-- Navigation and page updates-->
        <div>
    </div>

    <div id="content" class="page-content">
       <!-- Features of your page-->
    </div>
The strcture above will create the page. The first href will create the button to deploy the navigation, the first img class is the logo, the last href will redirect to the contact form. 
For features that can be added to the content page, please check the second documentation of your page. 
 

Page Navigation

<div class="sidebar-navigation">
    <a href="index.html"  class="sidebar-icon sidebar-1"></a>
    <a href="type.html"   class="sidebar-icon sidebar-2"></a>

    <div class="clear"></div>

    <a href="folio.html"   class="sidebar-icon sidebar-3"></a>
    <a href="#" class="close-sidebar sidebar-icon sidebar-4"></a>
    <a href="jquery.html"  class="sidebar-icon sidebar-5"></a>

    <div class="clear"></div>

    <a href="gallery.html"   class="sidebar-icon sidebar-6"></a>
    <a href="blog.html"    class="sidebar-icon sidebar-7"></a>
</div>

This is the page navigation, it can be found in the sidebar class. To edit the navigation simply change the hrefs locations. To change the icons, check the design folder, we've included the PSD to the navigation to edit it. 

You can use two navigations, further editing requires changing the menus-background.css or any of the css's you use in the style folder.
