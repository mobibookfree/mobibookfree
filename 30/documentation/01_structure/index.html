FAQ

How do I install the template?

Copy and paste the items in the code folder to your server

How do I customize the contact form?

Logo and social icons

    <div class="container header-under">
        <div class="logo-container">
            <img src="images/misc/logo.png" width="85">
            <strong>It's Compatible</strong>
        </div>

        <div class="social-container">
            <a href="#" class="twitter-header"></a>
            <a href="#" class="facebook-header"></a>
            <div class="clear"></div>
            <strong>Just Socials</strong>
        </div>
        <div class="clear"></div>
    </div>

 

Adding the above structure at the beginning of your content will create your logo and socials section. 

Replace the logo.png image in the misc folder with your logo to change the logo and also replace the width value with you're logo's width. Be sure that your image is for example 200 pixels and the value is only 100 (half) to keep a retina image! 

 

Page Structure

The page is structured into two main things

<div class="content">
    <div class="container">
        <!--Your Elements Go Here-->
    </div>
</div>

The content hosues the containers and creates the side margins, and the containers hold all the elements in the second documentation except for the image sliders! 

 

Top Structure

<div class="header">
    <div id="menu">
        <div id="menu-deploy">
            <a class="hide-menu" href="#"></a>
            <a class="show-menu" href="#"></a>
        </div> 

        <em class="header-message">
            Welcome
        </em>

        <a href="contact.html" class="contact-header"></a>        

        <div id="box">  
            <div class="box-deco"></div>  
            <a class="nav-item home-icon" href="index.html">Home</a>
            <a class="nav-item type-icon" href="type.html">Type</a>
            <a class="nav-item jque-icon" href="jquery.html">jQuery</a>
            <a class="nav-item clos-icon" href="#">Close</a>
        </div>
    </div>
</div>
Copying the above structure right before opening  your content class will generate the entire header.
 
hide-menu and show-menu will create the top left closing and opening navigations
 
header-message is the top center text shown on the page 
 
contact-header is the icon on the top right side
 
The box houses the navigation elements, copy and paste a href, and give it a unique class name like home-icon type-icon, in the style.css file line 70 you'll find the code that creates the icons. You can use more icons found in the images/icons folder to create your own navigation elements.
